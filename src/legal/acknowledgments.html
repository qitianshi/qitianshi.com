Typefaces

Grape Nuts is the font family used in heading text. A modified version is used in some branding and other imagery. Grape Nuts was designed by Robert Leuschke, and is licensed from Google Fonts under the Open Font License.

Barlow is the font family used in body text. It was designed by Jeremy Tribby, and is licensed from Google Fonts under the Open Font License.

Technologies

The source code for this website is written in HTML, Sass, and JavaScript. Webpack is used to bundle JavaScript files, and Jekyll is used to generate the static site. GitHub Actions is used for CI/CD, and the website is hosted on GitHub Pages.
